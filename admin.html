TheStyleDiaries Admin Panel — One-Time Setup
Step 1 — GitHub Personal Access Token

Yeh token admin panel ko permission deta hai GitHub mein posts save karne ki. Ek baar set karo — phir kabhi nahi karna.

Token banane ke liye:

1. github.com/settings/tokens/new kholo

2. Note: "TheStyleDiaries Admin" likho

3. Expiration: "No expiration" choose karo

4. Scopes: sirf "repo" checkbox tick karo

5. "Generate token" click karo → copy karo (ghps_ se shuru hoga)

Yeh sirf tumhare browser mein save hoga — kisi ko share mat karna!
Step 2 — Repository Details

Tumhara GitHub username aur repository name daalo.

Yeh password admin panel login ke liye hoga.

How This Works — Samjho

Posts kahan save hote hain? Tumhare GitHub repository mein — posts/ folder mein. Har post ek JSON file hai.

Website kaise update hoti hai? GitHub update hote hi Netlify automatically detect karta hai aur 1-2 minute mein website deploy ho jaati hai.

Kisi bhi device se kaam? Haan! Admin panel URL tumhara netlify-site/admin.html hai — phone, laptop, anywhere se kholo, GitHub se sab data milega.

Backup? GitHub khud backup hai — sab posts GitHub par permanently safe hain.
